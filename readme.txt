== Description ==

**Secure Signups: Strengthen Your WordPress User Registration with Domain Whitelisting**

Secure Signups is a powerful WordPress plugin designed to enhance your site's security by restricting user registrations to approved domain emails. Perfect for preventing spam, this plugin allows administrators to easily manage a whitelist of domains directly from the admin panel. Effortlessly add, update, and toggle domain statuses to control who can sign up.

With Secure Signups, you can customize messages displayed during registration, ensuring clear communication with prospective users. The plugin also offers straightforward activation and deactivation, giving you seamless control over its functionality at any time.

Protect your WordPress site from unwanted registrations by whitelisting trusted domains with Secure Signups. Say goodbye to spam and enjoy a more secure user registration process.

### Secure Signups
- Domain-Based Registration: Enable registration solely for email addresses associated with specified domains.
- Admin Panel Management: Easily add, update, and toggle the status (active/inactive) of approved domain lists.
- Customizable Messages: Configure personalized messages to be displayed during the registration process.
- Plugin Control: Manage plugin functionality directly from the Plugins settings.
- Enable/Disable Option: Control the operational status of the plugin with a simple toggle while installed on your WordPress site.

== Installation ==

**Secure Signups [Installation Guide]**

1. You can:
Upload the entire `secure signups` folder to the `/wp-content/plugins/` directory via FTP.
or
Upload the zip file of plugin via *Plugins -> Add New -> Upload* in your WordPress Admin Panel.
or
Search **Secure Signups** in the search engine available on *Plugins -> Add New* and press *Install Now* button.
2. Activate the plugin through *Plugins* menu in WordPress Admin Panel.
3. Click on *Secure Signups*  menu to configure the plugin.
4. Ready, now you can use it.

== Frequently Asked Questions ==

= How does Secure Signups enhance security for user registrations? =
Secure Signups restricts registrations to specific email domains, ensuring that only users from approved domains can sign up on your WordPress site. This reduces the risk of unauthorized access and enhances overall site security.

= Can I manage the list of approved domains from the plugin settings? =
Yes, the plugin provides an intuitive admin panel interface where you can effortlessly add, update, and toggle the status (active/inactive) of the approved domain lists.

= Is it possible to customize the messages displayed during the registration process? =
Absolutely! Secure Signups allows users to craft personalized messages that will be shown to users attempting to register with email addresses from approved domains.

= Can I control whether the plugin functions on my site? =
Yes, the plugin offers control settings that enable users to decide whether Secure Signups will be active or inactive while installed on the WordPress site.
